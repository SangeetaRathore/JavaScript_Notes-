What is the DOM?
DOM stands for Document Object Model. It is a programming interface that allows us to create, change, or remove
elements from the document. We can also add events to these elements to make our page more dynamic.
The DOM views an HTML document as a tree of nodes. A node represents an HTML element.We can access these elements in the document and make changes to them using JavaScript.
